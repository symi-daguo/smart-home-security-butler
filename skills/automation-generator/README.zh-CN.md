# 自动化生成器

智能家居 AI 安全管家的安全自动化场景生成器。

## 概述

自动化生成器分析安全检测结果，智能生成自动化建议以提升家庭安全。它创建修复自动化、响应程序和预防措施。

## 功能特性

- 常见安全场景的规则模板
- 基于现有自动化的模式生成
- 事件响应自动化生成
- 支持 Home Assistant、Node-RED 和 KNX 网关
- 质量评分和置信度指标
- 推荐前的安全验证
- 一键应用，含备份和回滚
- 安全差距分析

## 输出平台

| 平台 | 输出格式 |
|------|---------|
| Home Assistant | 自动化 YAML、脚本、蓝图 |
| Node-RED | 流 JSON、子流、函数节点 |
| KNX 网关 | 场景、工作流、触发器 |

## 推荐类型

| 类型 | 说明 |
|------|------|
| 修复 | 检测到问题的直接修复 |
| 预防 | 防止未来发生 |
| 增强 | 改进现有安全性 |
| 响应 | 事件响应自动化 |
| 监控 | 更好的检测/可见性 |

## 安装

此子技能已包含在智能家居 AI 安全管家中，无需单独安装。

## 配置

### 可选的环境变量

| 变量 | 说明 |
|------|------|
| `HA_TOKEN` | 用于应用 HA 自动化 |
| `NR_TOKEN` | 用于应用 Node-RED 流 |
| `KNX_TOKEN` | 用于应用 KNX 网关场景 |

### 质量阈值

```json
{
  "minConfidence": 0.7,
  "maxRecommendationsPerAlert": 5,
  "autoApplyEnabled": false,
  "backupBeforeApply": true,
  "safetyValidation": true
}
```

## 工作流程

```
安全告警 → 上下文分析 → 模板匹配
→ 自定义 → 安全验证 → 质量评分
→ 用户审核 → 应用 → 验证
```

## 安全特性

- 应用前的试运行验证
- 变更前自动备份
- 所有应用的推荐均可回滚
- 关键自动化始终需要手动确认
- 与现有自动化的冲突检测
